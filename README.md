# 🚀 Mit tc by Amos – TikTok User Info Scraper  

Want detailed info on any TikTok user without logging in or using API keys? **Mit tc by Amos** has you covered!

✅ **Fetch details like:**  
- Followers count 👥  
- Video count 🎥  
- Total likes ❤️  
- And more!

## 💻 Installation  

```bash
git clone https://github.com/amoskim318/MiktcTITOk.git
cd Clawk
chmod +x clawk.sh
./clawk.sh


---

⚡ Usage

./clawk.sh <@username>
# or
./clawk.sh [username]
